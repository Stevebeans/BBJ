Step 1: Ads.txt Compliance

In order to declare Taboola as an authorized seller, please visit the Ads.txt Records tab in Taboola Backstage and copy all the presented records to the ads.txt file in your root domain. If you have more than one site, you will find ads.txt records for each of your sites. Failure to add ALL of these lines to your ads.txt file can result in a significant reduction in revenue. Please contact your account manager if you have any questions about ads.txt.

Step 2: Consent Management Platform (CMP) Signals

This Step only applies if you have EU traffic and use a Consent Management Platform (CMP). If not, you can proceed to Step 3.

In order to remain compliant with GDPR and all applicable privacy laws, if you implement a CMP on your properties, but Taboola fails to receive a valid consent signal that indicates users' consent choices, then we will serve only contextually-based recommendations without use of personal data until the signals are properly received. This might happen if Taboola's script loads before your CMP renders.To ensure that Taboola properly receives a consent signal from your CMP and recommends personalised content for the highest performance, please make sure that your CMP script loads as early as possible before Taboola's script.

Step 3: Code Integration

For help implementing, please watch this instructional video.

Place this code in the <head> section:

<script type="text/javascript">
  window._taboola = window._taboola || [];
  _taboola.push({article:'auto'});
  !function (e, f, u, i) {
    if (!document.getElementById(i)){
      e.async = 1;
      e.src = u;
      e.id = i;
      f.parentNode.insertBefore(e, f);
    }
  }(document.createElement('script'),
  document.getElementsByTagName('script')[0],
  '//cdn.taboola.com/libtrc/bigbrotherjunkies-network/loader.js',
  'tb_loader_script');
  if(window.performance && typeof window.performance.mark == 'function')
    {window.performance.mark('tbl_ic');}
</script>

Place this code where you want the "Mid Article Reco Reel" widget to render:

<div id="taboola-mid-article-reco-reel"></div>
<script type="text/javascript">
  window._taboola = window._taboola || [];
  _taboola.push({
    mode: 'rec-reel-3n4-a',
    container: 'taboola-mid-article-reco-reel',
    placement: 'Mid Article Reco Reel',
    target_type: 'mix'
  });
</script>

Place this code at the end of your <body> tag:

<script type="text/javascript">
  window._taboola = window._taboola || [];
  _taboola.push({flush: true});
</script>
